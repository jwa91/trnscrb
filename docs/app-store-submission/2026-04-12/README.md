# App Store Submission Assets

This folder contains the screenshots used while preparing the first Mac App Store submission for `trnscrb` version `1.0` build `84` on `2026-04-12`.

- `screen1.png`, `screen2.png`, `screen3.png`: raw captures taken during submission prep
- `screen1-appstore.png`, `screen2-appstore.png`, `screen3-appstore.png`: trimmed to exact App Store Connect dimensions (`2560x1600`)

The App Store-sized files were generated from the raw captures and used for the submission flow in App Store Connect.
